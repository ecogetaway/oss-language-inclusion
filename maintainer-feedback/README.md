# Maintainer Feedback

This folder contains **raw maintainer input** (issues, short notes, linked artifacts).
Processed patterns are maintained in [`signals/maintainer-signals.md`](../signals/maintainer-signals.md).

## How to contribute

Open an issue or submit a PR with a short note. Public links are preferred for verification.

Use this format:

```text
Project:
Role:
Challenge:
What would help:
```
