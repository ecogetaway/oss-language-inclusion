# Case Study: RaunoT/plex-rewind — Portuguese — PR #354

## Repository
[RaunoT/plex-rewind](https://github.com/RaunoT/plex-rewind)

## Contribution Type
Pull Request

## Language
Portuguese (pt)

## What Happened
A contributor submitted a Portuguese language PR to Plex Rewind. The only visible comment on the PR was an automated stale bot warning. No human maintainer engaged with the contribution. The PR was closed unmerged after approximately 38 days.

## Timeline
- Opened: 2025-2026
- Closed: ~38 days later
- Days open: ~38

## Closing Reason
Bot-stale — no human engagement

## Maintainer Response
None — only automated stale bot comment

## Why This Matters
Portuguese is spoken by over 250 million people globally, with Brazil hosting one of the world's largest open-source developer communities. A Portuguese contributor who submits a language PR and receives only a stale bot warning before closure has no pathway to understand what they did wrong, what the project needs, or how to try again. The absence of human engagement is itself the structural gap.
